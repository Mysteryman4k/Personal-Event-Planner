## 📱 Overview

The **Personal Event Planner App** is an Android application developed for the SIT305 assignment. It allows users to organize upcoming events, trips, and appointments through a structured interface with local data persistence.

## ✨ Features

### Core CRUD Operations
| Operation | Description |
|-----------|-------------|
| **Create** | Add new events with Title, Category, Location, and Date/Time |
| **Read** | Dashboard displays all upcoming events automatically sorted by date |
| **Update** | Select and modify existing event details |
| **Delete** | Remove events you no longer wish to track |

### Additional Features
- ✅ **Room Database** - Local data persistence (data survives app restarts)
- ✅ **Navigation Component** - Bottom navigation bar with Fragments
- ✅ **Input Validation** - Prevents empty titles and past dates
- ✅ **User Feedback** - Toast messages for confirmations and errors
